# Milestone 1 - Sistema Solar Sandbox

## Objetivo
Permitir que um jogador controle uma frota e a mova entre planetas do Sistema Solar.

## Escopo
- Sistema Solar fixo
- Uma frota por jogador
- Movimento entre planetas
- Persistência
- Comunicação via WebSocket

## Fora de Escopo
- Combate
- Recursos
- Tecnologia
- Colonização
- Multiplayer

## Critérios de Aceitação
- Jogador conecta
- Recebe frota inicial
- Visualiza planetas
- Inicia movimento
- Movimento continua offline
- Movimento termina corretamente
